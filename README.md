# OP Tools (NeoForge)

OP Tools is a small NeoForge mod that adds overpowered tools, ores, custom enchantments, and more.

This is the **NeoForge** edition of OP Tools. The Fabric version lives at [OP-Tools-Fabric](https://github.com/jaymanh/OP-Tools-Fabric).

## Supported versions

| Minecraft | NeoForge | Mod version |
|-----------|----------|-------------|
| 1.21.1    | 21.1.234 | 1.8.3       |

## Features

### Foods and fuels

Super fuel and super meat are craftable. The fuel smelts an entire stack in a furnace, and the food fills hunger and saturation.

### Enchantments

- **Auto Replant** — replaces crops when broken
- **Mending** — available from the enchanting table
- **Tree Breaker** — breaks all logs in a tree
- **Auto-Repair** — gradually repairs items over time
- **Elemental enchantments** — fire, ice, ground, water, and lightning

## Development

Requirements: **Java 21**, Gradle (wrapper included).

```bash
./gradlew build      # build the mod JAR
./gradlew runClient  # run a dev client
./gradlew runServer  # run a dev server
```
